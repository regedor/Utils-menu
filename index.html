Utils-menu by regedor

Improve your terminal.

Collection of nice scripts alias and useful stuff. Makes it easy to have everything working. This project is not maintained any more. You can fin a really nice substitute at: http://regedor.github.com/polvo

Dependencies

You can't use it on Windows :p

Install

You want to install utils-menu? Run the following command and it will install itself for you (utils-menu will be installed in ~/.utils-menu) curl -L http://goo.gl/b2pmx > .utils-menu-install && bash ./.utils-menu-install && rm .utils-menu-install If you want to install utils-menu in a different directory, it's easy! For example, to install utils-menu at ~/.utils-menu, just run: curl -L http://goo.gl/b2pmx > .utils-menu-install && UTILS_PROJECT_PATH=~/.utils-menu bash ./.utils-menu-install && rm .utils-menu-install
